3. K-means
	데이터를 유사한 그룹으로 묶는 것

1) 동작과정
	클러스터 중심점을 임의로 선택(k = 사용자가 지정한 클러스터 수)
	각 데이터 포인트를 가장 가까운 중심점에 할당
	클러스터에 속한 데이터 포인트들의 평균을 계산하여 새로운 중심점을 갱신
	중심점이 더 이상 변경되지 않을 때까지 (2), (3)를 반복하면서 진행

2) 장점
	단순하면서도 효율적인 알고리즘
	대용량 데이터셋에서도 적용가능
	클러스터 수를 지정하기 쉽고 직관적이다

3) 단점
	클러스터의 모양이 원모양으로 가정, 다른 모양의 클러스터를 잘 분리하지 못할 수 있다
	초기 중심점의 위치에 따라 결과가 달라질 수 있으므로 초기화 방법에 따른 영향을 받을 수 있다

4) 공식
	중심점 계산 : 해당 클러스터에 속한 데이터 포인트의 평균으로 계산
	중심점 = 클러스터/1 데이터 포인트의 개수

	데이터 포인트 할당 : 각 데이터 포인트는 가장 가까운 중심점에 할당,
			거리를 기반(유클리드거리)
	거리(x, m) = n은 데이터 포인트의 차원수, xj는 데이터 포인트의 x의 j번째 차원의 값
